Release History

Track the evolution of the fastest UI framework on Roblox.

v0.1.0
The Buffer Engine Update
Latest Release
  • Massive Rewrite: Migrated the entire core to use raw buffer heap memory allocations. Say goodbye to heavy Instance overhead.
  • Virtual CPU Bridge: Implemented a highly optimized `RenderStepped` loop that acts as a Virtual CPU to map byte data to Roblox's hardware bridge.
  • Memory Protection: Locked down the core API utilizing metatable spoofing and proxies (`newproxy(true)`) to prevent accidental memory corruption.
  • Added `Mittens.List` mapping to `UIListLayout` with bit32 packed alignment configurations.
  • Added `Mittens.AspectRatio` support.
